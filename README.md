# Digital gate

### system requirement

-   php8.2
-   nodejs 21
-   mysql

### installation guide

```bash
# install the project
composer install && npm install

# run migrate
php artisan migrate

# run db seeder
php artisan db:seed

# build frontend

npm run dev
# open new terminal
php artisan serve

```
